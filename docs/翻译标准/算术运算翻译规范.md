# 算术运算 — COBOL → Java 翻译标准

> 本文件定义 `ADD`、`SUBTRACT`、`MULTIPLY`、`DIVIDE`、`COMPUTE` 的翻译规范。
> 核心思路：COBOL 算术支持 BigDecimal（S9(15)V9(2)）和整型（PIC 9(4)），Java 分别对应 BigDecimal 和 int/long。

---

## 一、类型判定

| COBOL PIC | Java 类型 | 说明 |
|-----------|-----------|------|
| `PIC 9(n)` — n ≤ 9 | `int` | 整数 |
| `PIC 9(n)` — n > 9 | `long` | 超长整数 |
| `PIC S9(n)V9(m)` | `BigDecimal` | 含小数位 |
| `PIC S9(n) COMP-3` | `BigDecimal` | 压缩十进制 |
| `PIC 9(n)V9(m)` | `BigDecimal` | 无符号小数 |

---

## 二、ADD

```cobol
ADD a TO b                     |  b = b + a;
ADD a TO b GIVING c            |  c = b + a;
ADD a b TO c GIVING d          |  d = c + a + b;
```

BigDecimal 字段使用 `.add()`：

```java
// COBOL: ADD 1 TO WSAA-TOTAL  (WSAA-TOTAL is BigDecimal)
wsaaTotal = wsaaTotal.add(BigDecimal.ONE);
```

---

## 三、SUBTRACT

```cobol
SUBTRACT a FROM b              |  b = b - a;
SUBTRACT a FROM b GIVING c     |  c = b - a;
```

---

## 四、MULTIPLY

```cobol
MULTIPLY a BY b                |  b = b * a;
MULTIPLY a BY b GIVING c       |  c = b * a;
```

---

## 五、DIVIDE

```cobol
DIVIDE a INTO b GIVING c       |  c = b / a;
DIVIDE a BY b GIVING c         |  c = a / b;
```

BigDecimal 除法使用 `a.divide(b, 2, RoundingMode.HALF_UP)`。

---

## 六、COMPUTE

### 1）整型

```cobol
COMPUTE WSAA-COUNT = WSAA-COUNT + 1
```
```java
wsaaCount = wsaaCount + 1;
```

### 2）BigDecimal

```cobol
COMPUTE WSAA-TOTAL = WSAA-AMOUNT * 2
```
```java
wsaaTotal = java.math.BigDecimal.valueOf(wsaaAmount.doubleValue() * 2);
```

使用 `BigDecimal.valueOf(double)` 包装计算结果，保持类型一致。

### 3）运算符

| COBOL | Java |
|-------|------|
| `+` | `+` |
| `-` | `-` |
| `*` | `*` |
| `/` | `/` |
| `( )` | `( )` |

---

## 七、INITIALIZE

| COBOL | Java |
|-------|------|
| `INITIALIZE WSAA-PARAMS` | `wsaaParams = new WsaaParams();` |
| `INITIALIZE WSAA-COUNT` | `wsaaCount = 0;` |
| `INITIALIZE WSAA-NAME` | `wsaaName = "";` |

结构体 INITIALIZE 重置为 new 对象；标量重置为类型默认值（0 或 ""）。
