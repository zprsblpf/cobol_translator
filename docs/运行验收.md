# 运行与验收

本文档记录当前仓库的最小可复现运行方式。项目分为两类路径：

- 确定性核心路径：解析、预处理、变量解析、ASG/visitor、规则翻译、骨架生成；不依赖 LLM。
- 全量翻译路径：LangGraph 编排、LLM 兜底、可选 RAG/Mongo；需要安装额外依赖和本地或远端 OpenAI 兼容模型服务。

## 1. 安装依赖

建议使用虚拟环境：

```bash
python -m venv .venv
.venv/Scripts/activate
python -m pip install -r requirements.txt
```

Windows PowerShell 可用：

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
```

## 2. 最小验收

不需要 LLM、不需要 MongoDB：

```bash
python scripts/check.py
```

默认等价：

```bash
python scripts/check.py --suite all
```

该命令会执行：

- Python 语法编译检查。
- 全量 unittest 回归。
- 使用 `tests/fixtures/minimal.cob` 跑 `main.py --parse-only`。
- 使用 `tests/fixtures/minimal.cob` 跑 deterministic skeleton 与 `main.py --no-llm` 输出检查。

当前基线：221 个测试通过，2 个本地 vLLM 测试在模型服务不可用时跳过。

并行开发 focused suite：

```bash
python scripts/check.py --suite quick
python scripts/check.py --suite leaf
python scripts/check.py --suite asg
python scripts/check.py --suite deterministic
python scripts/check.py --suite all
```

- `quick`：Python 语法编译、`test_smoke`、minimal parse-only。
- `leaf`：`test_translation.TestLeaf*` 与 `TestUnifiedLeafEntry`。
- `asg`：`test_translation.TestAsg*` 与 `TestDiffAsgVsLegacy*`。
- `deterministic`：minimal skeleton 生成与 `main.py --no-llm` 输出。
- `all`：完整本地闸口；也是 `python scripts/check.py` 默认行为。

## 3. 单独运行测试

```bash
python -m unittest -v
```

只跑最小 COBOL smoke：

```bash
python -m unittest test_smoke -v
```

## 4. 解析 COBOL

```bash
python main.py tests/fixtures/minimal.cob --parse-only
```

## 5. 生成确定性骨架

不走 LLM 的整程序翻译主入口：

```bash
python main.py tests/fixtures/minimal.cob --no-llm --output output
```

这会输出单个 Java 文件，例如 `output/Minismoke.java`。该路径不导入 LangGraph，不调用 vLLM；规则覆盖不了的复杂语句会保守输出 TODO。

底层骨架脚本也可以直接使用：

```bash
python scripts/translate_skeleton.py --in tests/fixtures/minimal.cob --out output/Minismoke.java
```

## 6. 全量翻译

全量翻译依赖 `langgraph` 和 LLM 相关依赖。默认模型端点是：

```text
http://localhost:8000/v1
```

可通过环境变量切换：

```bash
LLM_BASE_URL=http://localhost:8000/v1
LLM_API_KEY=dummy
LLM_MODEL=qwen3-8b-GPTQ
```

运行：

```bash
python main.py your_program.cob --output output
```

快速限定前 N 个 SECTION：

```bash
python main.py your_program.cob --sections 5 --output output
```

## 7. 当前注意事项

- `requirements.txt` 记录了全量路径需要的依赖；当前核心单测不强制导入所有可选依赖。
- 本地 vLLM 不在线时，`test_translation.TestLocalModel` 下的模型联通测试会跳过。
- `output/`、`logs/`、`chroma_db/`、`.env`、本地 Claude 设置和 stackdump 已被 `.gitignore` 忽略。
