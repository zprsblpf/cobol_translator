# 步骤36-组字段赋值覆盖设计

## 状态

已实现。

## 目标

补强 `INITIALIZE` 和组字段赋值覆盖，尤其是已知结构前缀下的单字段初始化，避免 `INITIALIZE WSAA-DATA` 这类语句不必要地 fallback。

## 规则

- `INITIALIZE X-PARAMS`：结构整体重置为 `new XParams()`。
- `INITIALIZE X-FIELD`：当 `X` 是已知结构前缀且字段不是调用约定字段时，生成 setter 默认值。
- 已知普通字段：沿用原有 `int/long -> 0`、`BigDecimal -> BigDecimal.ZERO`、字符串 -> `""`。
- `REPLACING` 等复杂 INITIALIZE 子句继续 fallback。
- `FUNCTION`、`FORMAT`、`STATUZ` 这类调用约定字段不在 leaf 层臆造初始化。

## 实现点

- `translator/leaf/assign.py`：扩展 `_t_initialize`，复用 `_assign` 输出结构字段 setter。
- `test_translation.py`：补充结构字段初始化、多字段初始化和复杂子句 fallback 测试。
