## 拆分原则

- 按泳道拆分：产品主线、unsupported、coverage、rules asset、leaf rules、flow/data、verification。
- 每个任务包声明允许修改文件和禁止修改文件。
- 第一批优先铺底座，第二批再并行补叶子规则。
- 每个 agent 只执行一个任务包。
