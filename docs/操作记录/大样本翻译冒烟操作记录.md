# 大样本翻译冒烟操作记录

## 目标

为 `scripts/translate_skeleton.py` 增加大样本快速冒烟能力（`--limit` 和 `--stats` 参数），
更新 `docs/运行验收.md` 的验收说明。

## 修改

| 文件 | 修改 |
|------|------|
| `scripts/translate_skeleton.py` | 增加 `--limit N`（限制前 N 段）+ `--stats`（详细统计：TODO 计数、叶子 TODO、LLM 调用） |
| `docs/运行验收.md` | §5.1 新增大样本翻译冒烟命令说明 |

## 验证

```powershell
python scripts/translate_skeleton.py --in tests/fixtures/minimal.cob --out tmp/minimal_java.txt --limit 1 --stats
```

输出包含：
- `SECTION 数: 2 (限制: 1)`
- `已翻卷段数: 1`
- `TODO 总数: 1`
- `LLM 调用: 1`

`scripts/check.py --suite all` 无回归。
