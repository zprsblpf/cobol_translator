## 4. 已知限制 / 范围外
- `MOVE WRITE TO X-FUNCTION` 裸写形态因 `WRITE` 与 COBOL 动词撞名被 segmenter 误切；真实 LIFE 写码用 5 字缩写（WRITR/WRITS/REWRT/UPDAT/DELET）不撞名。属 segmenter 既有限制，非本步骤范围。
- 批量写/循环内写、LLM 兜底链路不在本步骤；散点写 CALL 仍由叶子 `_t_call` 按 operations 兜底。
