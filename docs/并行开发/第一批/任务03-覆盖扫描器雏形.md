# 任务03：覆盖扫描器雏形

## 目标

新增规则覆盖扫描器，统计真实 COBOL 中各动词 supported/unsupported 数量，为后续补规范提供工作台。

## 泳道

C：Coverage

## 允许修改

- `scripts/scan_rule_coverage.py`
- `test_smoke.py`
- `tests/fixtures/*`
- `docs/操作记录/步骤35-覆盖扫描器雏形操作记录.md`

## 禁止修改

- `translator/leaf/*` 具体翻译逻辑
- `asg/visitor.py`
- `graph/nodes.py`

## 依赖

- 可独立启动。
- 若任务02已合并，则读取 structured unsupported；否则先做动词级统计。

## 实施步骤

1. 新增脚本 `scripts/scan_rule_coverage.py`。
2. 使用 `parser.cobol_parser.parse` + ASG 或 segmenter 统计语句首动词。
3. 对 leaf 语句调用 `translate_leaf_stmt` 判断 supported。
4. 输出终端摘要。
5. 输出 JSON/Markdown 到指定目录或 stdout。
6. 增加 smoke 测试，覆盖 minimal fixture。
7. 更新操作记录。

## 验收命令

```bash
python scripts/scan_rule_coverage.py tests/fixtures/minimal.cob
python -m unittest test_smoke -v
python scripts/check.py
```

## 完成标准

- minimal 可生成覆盖报告。
- 报告包含 supported/unsupported 计数。
- 不改变翻译产物。

## 合并备注

可与任务01、任务02、任务04 并行。
