# 任务04：快速验证入口

## 目标

给并行开发 agent 提供 focused check suite，减少每次开发都跑全量测试的等待成本。

## 泳道

G：Verification

## 允许修改

- `scripts/check.py`
- `test_smoke.py`
- `docs/运行验收.md`
- `docs/操作记录/步骤36-快速验证入口操作记录.md`

## 禁止修改

- 翻译语义代码：`translator/*`, `asg/*`, `parser/*`

## 依赖

- 无。

## 实施步骤

1. 为 `scripts/check.py` 增加 `--suite quick|leaf|asg|deterministic|all`。
2. `quick` 跑 py_compile、smoke、minimal parse。
3. `leaf` 跑 `TestLeaf*` 和 `TestUnifiedLeafEntry`。
4. `asg` 跑 `TestAsg*` 和 `TestDiffAsgVsLegacy*`。
5. `deterministic` 跑 minimal no-LLM/deterministic 输出。
6. `all` 等价当前全量本地闸口。
7. 增加不执行子进程的 suite 配置测试。
8. 更新运行验收文档。

## 验收命令

```bash
python scripts/check.py --suite quick
python scripts/check.py --suite leaf
python scripts/check.py --suite asg
python scripts/check.py --suite deterministic
python scripts/check.py --suite all
```

## 完成标准

- 所有 suite 可运行。
- `python scripts/check.py` 默认仍保持全量本地闸口或明确等价 `--suite all`。

## 合并备注

建议第一批优先合并，便于后续 agent 使用 focused checks。
